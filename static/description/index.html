How to Install and Configure Duitku Payment

Installation and Activating

To install Duitku Payment in Odoo, follow these steps:

  1. Download the Duitku Payment module.
  2. Place the folder in the addons path . Rename the folder as payment_duitku.
  3. Restart Odoo to recognize the new module.
  4. Activate the module in the Apps menu.

Configuring Duitku Payment for Sales

Before activating Duitku Payment, ensure that the Sales Apps is installed. Once installed, follow these steps:

  1. Navigate to Payment Providers:
    • Click the square icon in the top-left corner and select Sales.
    • Go to Configuration > Online Payments > Payment Providers.
  2. Enable Duitku Payment:
    • Click on Duitku Payment to open its configuration.
    • Set State to Enable for Production or Test Mode for sandbox.
    • Enter the Merchant Code and API Key corresponding to the selected mode.
  3. Activate Payment Methods:
    • From Sales Apps, go to Configuration > Online Payments > Payment Methods.
    • Ensure Duitku Payment is active so it appears as a payment option for clients.

Configuring Duitku Payment for E-commerce

Before activating Duitku Payment, ensure that the E-commerce Apps is installed. Once installed, follow these steps:

  1. Navigate to Payment Providers:
    • Click the square icon in the top-left corner and select Website.
    • Go to Configuration > eCommerce > Payment Providers.
  2. Enable Duitku Payment:
    • Click on Duitku Payment to open its configuration.
    • Set State to Enable for Production or Test Mode for sandbox.
    • Enter the Merchant Code and API Key corresponding to the selected mode.
  3. Activate Payment Methods:
    • From Sales Apps, go to Configuration > eCommerce > Payment Methods.
    • Ensure Duitku Payment is active so it appears as a payment option for clients.

Configuring Duitku Payment for Point of Sale (POS)

Before activating Duitku Payment, ensure that the E-Point of Sale Apps is installed. Once installed, follow these steps:

  1. Navigate to Payment Providers:
    • Click the square icon in the top-left corner and select Point of Sale.
    • Go to Configuration > Payment Methods.
  2. Create a New Payment Method:
    • Name the method Duitku Payment (Possible use another name such as "Online Payments").
    • Check Online Payment.
    • In Allowed Providers, select Duitku Payment.
  3. Verify Activation:
    • Ensure the new payment method appears in the Payment Methods List.
    • If missing, go to Configuration > Settings.
    • Select the relevant POS and confirm Duitku Payment is listed under Payment Methods.
    • If not, manually add it.
By following these steps, Duitku Payment will be successfully installed and configured in Odoo for Sales, E-commerce, and POS. Click here to see the integration tutorial on Youtube!
