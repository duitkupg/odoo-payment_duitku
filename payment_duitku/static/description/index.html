Installation

For installation to the Odoo you might download this code. Then, drop the folder to the addons path, then rename it as payment_duitku. After it's done you may follow steps on below.

  • Go to your Apps list on Odoo.
  • Find Duitku Payment Acquirer. Then, click install.
  • Go to the Website.
  • On the Website menu go to the Configuration -> eCommerce -> Payment Acquirers.
  • On the list Payment Acquirers click on Duitku Payment.
  • Edit the payment and fill with your Duitku credentials. (If you don't have any credentials you might start from creating project on Duitku Dashboard).
  • Click save and done.
You've finish the installation process. You might see the Duitku payment on the checkout payment list.
